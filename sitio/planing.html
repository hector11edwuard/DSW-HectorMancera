GIT Y GITHUB

Sistema control de versiones

Objetivo del Sitio

Comprender las características principales de GIT y GITHUB, para la administración y control de versiones de nuestros proyectos de desarrollo.

Objetivos Específicos

1- Instalar Git y GitHub como sistema control de versiones.

2- Administrar los repositorios del sistema de control de versiones Git y GitHub.

3- Identificar las funciones principales de Git y GitHub.

Materiales

  • Pagina de GIT
  • Pagina de GITHUB

Contenido

Bienvenido a mi sitio web del curso de Diseños de sitios Web de la UNAD, este sitio trata temas específicos de Git y GitHub, donde se dará información con respecto a estos temas, como la instalación y administración de la herramienta, se explicarán sus características y funciones principales como sistema control de versiones. Se mostrará la maquetación del sitio con el fin de irlo adecuando con el mejor diseño.
